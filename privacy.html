Politique de Confidentialité

Dernière mise à jour : 14 octobre 2025

1. Introduction

Bienvenue sur BibleShot. Nous respectons votre vie privée et nous nous engageons à protéger vos données personnelles. Cette politique de confidentialité vous informe sur la manière dont nous traitons vos données personnelles lorsque vous utilisez notre application mobile.

2. Données que nous collectons

2.1 Données collectées automatiquement

BibleShot est conçue pour minimiser la collecte de données. L'application stocke localement sur votre appareil :

  • Vos préférences de thème visuel
  • Vos statistiques de jeu (quiz joués, scores, séries)
  • Vos shots favoris
  • L'historique de vos quiz
  • Vos paramètres de notifications

2.2 Données d'analyse anonymisées

Pour améliorer l'application et comprendre comment elle est utilisée, nous collectons des données d'usage anonymisées via Mixpanel :

  • Actions effectuées dans l'app (quiz joués, shots consultés, etc.)
  • Informations techniques (version de l'app, modèle d'appareil, version iOS)
  • Données de performance et de stabilité de l'application
  • Un identifiant anonyme unique (non lié à votre identité personnelle)

2.3 Données que nous ne collectons PAS

  • Aucune donnée personnelle identifiable (nom, email, adresse)
  • Aucune donnée de localisation précise
  • Aucun contenu de vos réponses aux quiz
  • Aucune information financière (gérée directement par Apple)

3. Utilisation de vos données

3.1 Données locales

Les données stockées localement sur votre appareil sont utilisées uniquement pour :

  • Personnaliser votre expérience (thème, notifications)
  • Sauvegarder votre progression et vos statistiques
  • Conserver vos shots favoris

Ces données personnelles ne sont jamais transmises à nos serveurs.

3.2 Données d'analyse (Mixpanel)

Les données anonymisées collectées via Mixpanel sont utilisées pour :

  • Comprendre comment l'application est utilisée et améliorer l'expérience utilisateur
  • Identifier et résoudre les problèmes techniques
  • Mesurer l'engagement et les fonctionnalités les plus appréciées
  • Optimiser les performances de l'application

Ces données sont agrégées et anonymes. Elles ne permettent pas de vous identifier personnellement.

4. Stockage et sécurité

4.1 Stockage local

Vos données personnelles (préférences, statistiques, favoris) sont stockées localement sur votre iPhone via AsyncStorage (technologie sécurisée d'Apple). Ces données sont protégées par les mécanismes de sécurité natifs d'iOS.

4.2 Données d'analyse

Les données anonymisées sont transmises de manière sécurisée à Mixpanel via HTTPS. Mixpanel est certifié conforme au RGPD et respecte les normes internationales de protection des données. Pour plus d'informations : Politique de confidentialité de Mixpanel.

5. Notifications

Si vous activez les notifications :

  • Nous utilisons les services de notification d'Apple (APNs)
  • Les notifications sont envoyées localement par votre appareil
  • Aucun identifiant personnel n'est collecté
  • Vous pouvez désactiver les notifications à tout moment dans les paramètres de l'app

6. Partage de données

BibleShot partage uniquement des données anonymisées avec les partenaires suivants :

  • Mixpanel : Service d'analyse qui reçoit des données d'usage anonymisées pour nous aider à améliorer l'application
  • Apple : Pour la gestion des achats et abonnements via l'App Store

Vos données personnelles (préférences, statistiques, favoris) ne sont jamais partagées avec des tiers. Seules des données d'usage anonymisées et agrégées sont transmises à Mixpanel.

Vous pouvez également choisir de partager manuellement des shots via les outils de partage iOS (Messages, réseaux sociaux, etc.).

7. Vos droits

Conformément au RGPD et aux lois sur la protection des données, vous avez le droit de :

  • Accéder à vos données : Toutes vos données locales sont visibles dans l'application
  • Modifier vos données : Changez vos préférences dans les paramètres
  • Supprimer vos données locales : Désinstallez l'application ou réinitialisez-la dans les paramètres
  • Refuser le tracking : Vous pouvez désactiver le suivi des données d'analyse dans les réglages iOS (Confidentialité > Suivi) ou nous contacter pour demander la suppression de vos données Mixpanel
  • Portabilité : Vos données personnelles restent sur votre appareil

8. Conservation des données

8.1 Données locales

Vos données personnelles sont conservées localement sur votre appareil jusqu'à ce que vous :

  • Supprimiez l'application
  • Réinitialisiez vos données dans les paramètres de l'app
  • Réinitialisiez votre appareil

8.2 Données d'analyse

Les données anonymisées collectées par Mixpanel sont conservées conformément à leur politique de rétention (généralement 5 ans maximum). Vous pouvez demander la suppression de ces données en nous contactant.

9. Données des mineurs

BibleShot est accessible à tous les âges (classification 4+). Nous ne collectons aucune donnée personnelle identifiable. Les seules données collectées sont des informations d'usage anonymisées qui ne permettent pas d'identifier les utilisateurs, y compris les mineurs.

10. Cookies et technologies de suivi

L'application mobile n'utilise aucun cookie car elle ne fonctionne pas dans un navigateur web.

Nous utilisons Mixpanel comme service d'analyse pour collecter des données d'usage anonymisées. Mixpanel utilise un identifiant unique anonyme pour suivre les événements dans l'application, mais cet identifiant n'est pas lié à votre identité personnelle.

Vous pouvez contrôler le suivi des données d'analyse :

  • Via les réglages iOS : Réglages > Confidentialité > Suivi
  • En nous contactant pour demander l'opt-out complet de Mixpanel

11. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. En cas de modification importante, nous vous en informerons via une mise à jour de l'application. La date de "dernière mise à jour" en haut de cette page sera modifiée.

12. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter :

  • Email : contact@bibleshot.fr
  • Via le formulaire de contact : Page de support

13. Conformité RGPD

BibleShot respecte le Règlement Général sur la Protection des Données (RGPD) :

  • Minimisation des données : Nous collectons uniquement les données strictement nécessaires à l'amélioration de l'application
  • Anonymisation : Les données d'analyse sont anonymisées et ne permettent pas de vous identifier
  • Consentement : Vous êtes informé de la collecte de données et pouvez refuser le tracking
  • Transparence : Cette politique explique clairement nos pratiques de collecte et d'utilisation des données
  • Sécurité : Vos données personnelles sont protégées par iOS, les données d'analyse sont transmises via HTTPS
  • Droits des utilisateurs : Vous pouvez accéder, modifier et supprimer vos données à tout moment
  • Portabilité : Vos données personnelles restent sur votre appareil

13.1 Base légale du traitement

  • Données locales : Nécessaires à l'exécution du contrat (fourniture du service)
  • Données d'analyse : Intérêt légitime à améliorer notre service, avec possibilité d'opposition

En résumé : BibleShot collecte uniquement des données d'usage anonymisées via Mixpanel pour améliorer l'application. Aucune donnée personnelle identifiable n'est collectée. Toutes vos informations personnelles (préférences, statistiques, favoris) restent sur votre iPhone et ne sont jamais transmises à nos serveurs.
